## 📋 О проекте

**Vertex** — это веб-приложение для автоматизированного анализа финансовой отчётности компаний. Система принимает PDF-документы, извлекает ключевые финансовые показатели с помощью LLM, рассчитывает коэффициенты, оценивает риски и формирует инвестиционные рекомендации.

### 🔹 Основные возможности

| Функция | Описание |
|---------|----------|
| 📊 **Извлечение метрик** | Автоматическое извлечение выручки, прибыли, активов и других показателей из PDF |
| 📈 **Финансовые коэффициенты** | Расчёт 10+ коэффициентов (ликвидность, рентабельность, устойчивость, деловая активность) |
| 🎯 **Интегральный скоринг** | Оценка компании по 100-балльной шкале на основе финансовых показателей |
| 🧠 **NLP-анализ** | Выявление рисков и возможностей из текстовой части отчёта |
| 💡 **Рекомендации** | Генерация структурированных рекомендаций на основе данных |
| 🎨 **Веб-интерфейс** | Удобный UI для загрузки документов и визуализации результатов |

## 🚀 Быстрый старт

### Требования

- Python 3.11 или выше
- pip 22.0+
- Docker (опционально)

### Установка и запуск

#### Способ 1: Локально

```bash
# 1. Клонирование репозитория
git clone https://github.com/ma1neee/VertexAI.git
cd vertex

# 2. Создание виртуального окружения
python -m venv venv

# 3. Активация окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установка зависимостей
pip install -r requirements.txt

# 5. Настройка переменных окружения
cp .env.example .env
# Отредактируйте .env и добавьте ваши API-ключи

# 6. Запуск бэкенда
python src/app.py
# Или:
python -m uvicorn src.app:app --host 0.0.0.0 --port 8000 --reload

# 7. Запуск фронтенда (отдельный терминал)
cd frontend
python -m http.server 3000
```

#### Способ 2: Docker

```
# Сборка и запуск контейнеров
docker-compose up --build

# Остановка
docker-compose down
```
